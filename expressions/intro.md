# Expressions

An expression is a combination of operands and operators to produce results when evaluated. A simple example is shown below

```py
x = 1 + 1
print(x)
```

## Operator Precedence

The table below shows operators and their associativity

Binary ** Right
Unary +, -
Binary *, /, //, % Left
Binary +, - Left
Binary = Right
